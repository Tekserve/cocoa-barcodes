Methods


normalizeSize

Abstract: Normalize the resolution to 72 dpi in lossless fashion so that the image displays at full size onscreen. 
- (NSImage *)normalizeSize;

Result: NSImage with the exact same data as the instance it is called on except that the resolution pixelHeight will be the same as the height and the pixelWidth will be the same as the width.

setDPI:

Abstract: Allows you to adjust (in a lossless fashion) the resolution of the images NSBitmapImageRep (if it has one) to a given number of points per inch. 
- (NSImage *)setDPI:(int)dpi;

Parameters

NameDescription
dpiInteger value representing the desired horizontal and vertical pixels per inch.
Result: Self with the data unaltered except for the height and width (pixelHeight and pixelWidth remain unchanged) of the first NSBitmapRep the image contains.
