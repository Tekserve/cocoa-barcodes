generateChecksum

Abstract: Generates code 3 of 9 check digit based on content 
- (void)generateChecksum;

Code 39's optional checksum is rarely used, but we'll provide the option here. It's a fairly simple algorithm - it's the sum of the value (via lookup table) of each character and doing a modulus 43 on the sum, storing the result as a character in the field checkDigit
