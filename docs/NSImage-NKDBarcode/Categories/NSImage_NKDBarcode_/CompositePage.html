NSImage(NKDBarcode)


Discussion

This category adds a single class method to NSImage that allows you to generate a resolution-independent NSImage from an NKDBarcode object.

Methods

imageFromBarcode:

Abstract: Creates resolution independent image from barcode. 
+ (NSImage *)imageFromBarcode:(NKDBarcode *)barcode;

Parameters

NameDescription
barcodeAn NKDBarcode object that needs to be drawn.
Result: Initialized NSImage with PDF representation that represents the barcode pass in.
