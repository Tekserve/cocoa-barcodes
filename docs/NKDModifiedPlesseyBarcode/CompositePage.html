Discussion



This is an encoder for the Plessey Code. According to the Plessy Code Specification, numeric digits are assigned a set value, but the remaining values can be used for whatever values are desired. This subclass adds the ability to encode the letters A through F for full hexadecimal support.

Wide bar followed by narrow space is a "1" bit.
Narrow bar followed by wide space is a "O" bit.

We will use a 3:1 wide to narrow bar ratio.

THIS IS NOT GUARANTEED TO PRODUCE A BARCODE THAT IS IN COMPLIANCE WITH THE SPECIFICATION.
== USE AT YOUR OWN RISK ==



#defines

ONE_BIT

#define ONE_BIT @"1110"

Plessey encodes data in binary form. A '1' is encoded as a wide bar followed by a narrow space. For this framework, this is represented by the string "1110"

ZERO_BIT

#define ZERO_BIT @"1000"

Plessey encodes data in binary form. A '0' is encoded as a narrow bar followed by a wide space. For this framework, this is represented by the string "1000"
