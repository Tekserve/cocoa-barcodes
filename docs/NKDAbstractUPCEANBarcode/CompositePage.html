NKDAbstractUPCEANBarcode.h


Discussion

This is an abstract object to encapsulate the functionality that the various UPC and EAN barcodes have in common, including parity-swapping and digit encoding.

Typedefs

Handedness

typedef BOOL Handedness;

Defiens a type of Handedness (as a BOOL) to deal with the difference between left and right handed encoding schemes

Parity

typedef BOOL Parity;

Defines a type of "Parity" (as a BOOL) to make methods that take one of the precompiler defined parity values more readable.

#defines

EVEN_PARITY

#define EVEN_PARITY YES

UPC / EAN codes use parity swapping to increase the number of values that can be encoded. This defines EVEN_PARITY as YES to make the code more readable.
