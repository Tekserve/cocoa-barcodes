NKDExtendedCode39Barcode.h



Extended Code 3 of 9 uses the same encoding scheme as its parent Code 3 of 9, but it utilizes two-character combinations to expand the range of encodable characters to include lower-case alphabetical characters and more punctuation and diacriticals. This subclass of NKDCode39 implements this encoding scheme.
