NKDUPCEBarcode

Abstract: UPC-E abbreviated UPC symbology

This subclass can take one of three possible values - the actual UPC-E barcode (8 digits) value or the UPC-A barcode, either with or without the check digit. Either way, the code will print out as a UPC-E barcode. If the content value provided at time of instantiation is an actual UPC-E value, the check- digit MUST be encoded because UPC-E values do not contain enough information to accurately calculate the check-digit.
